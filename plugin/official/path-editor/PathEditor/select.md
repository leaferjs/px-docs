# 选择

## 关键方法

### select ( [`PathNode`](../PathNode.md) | [`PathNode`](../PathNode.md)[] )

选中节点元素。

### cancel ( )

取消选中节点元素。

### hasItem ( item: [`PathNode`](../PathNode.md) ): `boolean`

是否已选中某个节点元素。

### addItem ( item: [`PathNode`](../PathNode.md) )

新增一个节点元素到选中列表。

### removeItem ( item: [`PathNode`](../PathNode.md) )

从选中列表中移出节点元素。
