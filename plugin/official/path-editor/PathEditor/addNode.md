# 添加节点

## 关键方法

### addNode ( nodeData: [`IPathCommandNode`](../interface/PathCommandNodeData.md#ipathcommandnode) ): [`PathNode`](../PathNode.md)

添加一个节点到末尾。

### addNodeBefore ( nodeData: [`IPathCommandNode`](../interface/PathCommandNodeData.md#ipathcommandnode), before:[`PathNode`](../PathNode.md) ): [`PathNode`](../PathNode.md)

添加一个节点在指定节点元素的前面。

### addNodeAfter ( nodeData: [`IPathCommandNode`](../interface/PathCommandNodeData.md#ipathcommandnode), after: [`PathNode`](../PathNode.md) ): [`PathNode`](../PathNode.md)

添加一个节点在指定节点元素的后面。

### addNodeAt ( nodeData: [`IPathCommandNode`](../interface/PathCommandNodeData.md#ipathcommandnode), index: `number` ): [`PathNode`](../PathNode.md)

添加一个节点在指定位置。
