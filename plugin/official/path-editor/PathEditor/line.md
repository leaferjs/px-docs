# 线段操作

## 关键方法

### getLineNodes ( node: [`PathNode`](../PathNode.md) ): [`PathNode`](../PathNode.md)[]

获取节点所在线条的所有节点列表。

### spliceLineNodes (node: [`PathNode`](../PathNode.md) ): [`PathNode`](../PathNode.md)[]

splice 物理提取节点所在线条的所有节点列表。

### reverseLineNodes ( node: [`PathNode`](../PathNode.md) )

颠倒节点所在线条的所有节点前后顺序。

### deleteLineNodes ( node: [`PathNode`](../PathNode.md) )

删除节点所在线段的所有节点。
