# PxGrow AI

本仓库为 AI 系统提供 PxGrow 插件使用示例和文档，便于生成和理解代码。

## PxGrow

PxGrow（像素生长）是一个连接开发者与用户的软件生态平台，基于开源 LeaferJS 提供 AI 画布、设计工具与工业可视化解决方案，并通过插件与应用生态共同构建可运行、可交互、可编辑的实时数字世界。

请先阅读 [PxGrow 的介绍文档](./introduction.md)

## 目录说明

- **examples/**  
  独立可运行的示例，展示 PxGrow 各类插件的功能。  
  是 AI 学习模式和使用方法的主要来源。

- **plugin/official/**  
  PxGrow 官方插件文档。

- **introduction/**  
  PxGrow 每个插件的介绍及常见问题。

- **update/**  
  更新日志和版本记录（AI 可选读取）。

## AI 使用建议

生成 PxGrow 代码时：

1. 优先参考 `examples/` 中的简单示例。
2. 可以参考 `plugin/official/` 获取模式和 API 用法。
